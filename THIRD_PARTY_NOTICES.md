# Third-Party Performance Components

The S9Lab Client distribution includes unmodified Fabric mod JARs through
Fabric Loader's Jar-in-Jar mechanism. Copyright remains with their respective
authors. Source code and license terms are available from the linked projects.

- Sodium: PolyForm Shield 1.0.0 - https://github.com/CaffeineMC/sodium
- Lithium: LGPL-3.0-only - https://github.com/CaffeineMC/lithium-fabric
- FerriteCore: MIT - https://github.com/malte0811/FerriteCore
- EntityCulling: tr7zw Protective License - https://github.com/tr7zw/EntityCulling
- MoreCulling: GPL-3.0-only - https://github.com/fxmorin/moreculling
- Conditional Mixin 0.6.3 and MixinSquared 0.2.0: runtime libraries distributed with MoreCulling
- ImmediatelyFast: LGPL-3.0-or-later - https://github.com/RaphiMC/ImmediatelyFast
- Dynamic FPS: MIT - https://github.com/juliand665/Dynamic-FPS
- Battery 2.0.0: runtime library distributed with Dynamic FPS
- Cloth Config API: LGPL-3.0-only - https://github.com/shedaniel/ClothConfig

The embedded files are not modified by S9Lab Client. Their notices and license
files remain present inside their original JAR files.
